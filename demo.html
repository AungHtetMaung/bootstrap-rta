This RTA shows the default button set.

It is enabled by setting class="rta" on the textarea and making the following javascript call: 

$(document).ready(function() {
    $.rta();
});

The code above initializes all textarea's with an 'rta' class to have the default button set.

You can initialize a specific textarea by specifying a jquery selector as shown below:

$(document).ready(function() {
    $('#myTextArea').rta();
});

Two way editing (disabled by default) allows HTML source to be edited in the original textarea.

Click the HTML button above to switch between HTML and WYSIWYG editing.

There are 3 ways to enable two way editing:

  1. Add a data-rta-twoway="true" attribute to the textarea
  2. Pass the option as a parameter when initializing the RTA element(s) in JavaScript: $.rta({ 'twoway': "true" });
  3. Change the global option as follows: $.rta.options.twoway = true;

Note: When two way editing is disabled (the default), the original textarea (containing the HTML source) can still be viewed but it is marked read-only. See the 'All Buttons' example below for read-only behavior.

The buttons for this RTA are assigned in HTML via a data-rta="bold italic, ul ol" attribute on the textarea.

There are two ways to specify a custom set of buttons:

  1. Add a data-rta"bold italic, ul ol" attribute to the textarea (like this demo).
  2. Pass the button definition as a parameter when initializing the RTA element(s) in JavaScript: $.rta('bold italic, ul ol');

This RTA shows available edit buttons. The code to accomplish this follows:

$(document).ready(function() {
    $.rta($.rta.allButtons);
});

Note that there is an 'allButtons' property on the global 'rta' object that provides a convenient grouping of all available buttons.

The following code shows how to add all buttons to a specific textarea whose id="myTextArea"

$(document).ready(function() {
    $('#myTextArea').rta($.rta.allButtons);
});

This RTA upsizes from Bootstrap 'mini' buttons to 'small' buttons.

There are 3 ways to change the button size:

  1. Add a data-rta-btnsize="small" attribute to the textarea
  2. Pass the option as a parameter when initializing the RTA element(s) in JavaScript: $.rta({ 'btnsize': "small" });
  3. Change the global option as follows: $.rta.options.btnsize = 'small';

This is a plain textarea. This was included merely to show that you can mix RTA and plain textarea fields in the same form.
